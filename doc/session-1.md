# Session 1 Reflection

In this session I learned the basics of the Unix shell and Git. 
The shell allows us to navigate files and folders using commands such as cd, ls, and mkdir. 
We also learned how Git helps track changes in files and allows collaboration through GitLab. 
At first the workflow with cloning repositories and using Git commands was confusing, but after practicing I started to understand how it works. 
I think these tools will be useful for managing code and research projects.
