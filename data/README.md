# Datasets

* [pulse_data.csv](pulse_data.csv) contains timeseries with data from 
  pulse oxymetry sensor
* [Niederschlag_1981-2010_Stationsliste.txt](Niederschlag_1981-2010_Stationsliste.txt) list of weather stations with mean precipitation for the period 1981 to 2010, downloaded from [DWD](https://opendata.dwd.de/climate_environment/CDC/observations_germany/climate/multi_annual/mean_81-10)
* [Niederschlag_1981-2010.txt](Niederschlag_1981-2010.txt) precipitation data for the period 1981 to 2010, downloaded from [DWD](https://opendata.dwd.de/climate_environment/CDC/observations_germany/climate/multi_annual/mean_81-10)
* [produkt_precipitation_399_akt.txt](produkt_precipitation_399_akt.txt) hourly precipitation data for station 399 (Berlin, Alexanderplatz) dowloaded from [recent hourly data](https://opendata.dwd.de/climate_environment/CDC/observations_germany/climate_urban/hourly/precipitation/recent/) on 2025-04-28
